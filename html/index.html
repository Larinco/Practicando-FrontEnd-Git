Practicando Frontend Git

Descripción

Controlador de Versiones

Bloques

  • Working directory: Donde se hace el código de los archivos
  • Staging area: Donde se agregan los archivos para luego guardarlos, se crean versiones
  • Repository: Donde se guardan los archivos

Comandos Básicos de GIT

  • git init: usar git
  • git add file(entre signos de mayor y menor): añade los archivos del "working directory" al "staging area"
  • git status: verifica donde estan los archivos
  • git commit: añade los archivos del "staging area" al "repository", crea snapshots
  • git push: para subir los archivos a un repositorio remoto
  • git pull: traer archivos cambiados por otros desarrolladores
  • git clone: clona los archivos del código

Comandos de GIT Bash

  • ls: muestra directorias y archivos
  • pwd: muestra en que ruta del sistema se esta posicionando el terminal
